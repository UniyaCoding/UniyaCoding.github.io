Rules:

  • Berbicara sopan.

Tombol Keyboard:

  • Tombol Back Space / Delete: Untuk menghapus huruf.
  • Tombol Space: Untuk memberikan jarak di antara huruf.
  • Tombol Enter: Untuk memberi baris baru dan untuk setuju pada program yang ada.

Permainan Mengetik:

Ketiklah huruf yang ada untuk menyusun CUPS.

Link Permainan: Cup Stack Typing Game

Rocket Typing:

Ketiklah huruf yang ada di ROCKET untuk membuat nama elemen!

Link Permainan: Rocket Typing Junior

Menjadi Pemenang:

Ketiklah kata yang muncul secepat mungkin untuk menjadi pemenangnya!

Link Permainan: Sky Chase Typing
